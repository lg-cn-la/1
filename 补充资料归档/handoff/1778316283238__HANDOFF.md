### 用户对 v2 的真实反馈（关键）

1. **「v2 像 claude.com 复制」** — 暖纸 / 衬线 / 编辑式分栏正是 Anthropic / Stripe Press / Linear 改良 这一档"AI/SaaS 品味池"的中心。Anthropic 自己的 frontend-design skill 推荐的「Distinctive / Editorial」字体清单，反而把方向压回了"Anthropic 家味"——更深一层的 slop。

   - 没 artifact 内嵌预览 → 我每次写完没真打开看过，只校验过 HTML 标签平衡
   - plugin marketplace 拉取通道 bug（plugins.claude.ai DNS 不解析）→ design / frontend-design 真版本装不进来
   - headless 浏览器（Playwright/Puppeteer）沙箱里装不上 → 没法截图验证

## 4. 已应用的设计规则清单

来源：Anthropic 官方 frontend-design skill（已从公开 docs 抓出）

**禁用：**
- 字体：Inter、Roboto、Open Sans、Lato、Arial、系统字体、Space Grotesk（被滥用）
- 色板：白底紫渐变、温吞平均色板、纯白 / 纯黑底
- 结构：cookie-cutter 卡片网格、可预测布局

**必须：**
- 写代码前先确定**一个**美学方向（brutalist / maximalist / retro-futuristic / editorial / luxury / playful / industrial / brutalist 等）
- display × body 必须高对比配对（display + mono / serif + geometric sans / 可变字重）
- 字重用极端（200 vs 800，不是 400 vs 600）
- 字号跳跃 ≥3×（不是 1.5×）
- 背景做层次氛围（叠 radial / 几何 / 纹理）
- 一次性 stagger 入场动画 > 一堆碎微动效

**字体推荐池：**
- Code aesthetic: JetBrains Mono / Fira Code
- Editorial: Playfair Display / Crimson Pro / Fraunces
- Startup: Clash Display / Satoshi / Cabinet Grotesk
- Technical: IBM Plex 系列 / Source Sans 3
- Distinctive: Bricolage Grotesque / Obviously / Newsreader
